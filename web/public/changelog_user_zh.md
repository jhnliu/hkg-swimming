# 更新日誌

## 2026-03-22

### 新功能（最新）
- **學界游泳比賽成績** — 新增香港學界體育聯會中學校際游泳比賽成績，涵蓋9個賽季（2015/16 – 2024/25）、第一至三組，覆蓋港島、九龍及合併區域。超過65,000項成績，來自270多間學校，現已可搜尋
- **學界比賽頁面** — 瀏覽比賽、查看各項目初賽及決賽成績、學校獎牌及積分排名、最佳時間排行榜，以及各學校詳細頁面
- **學界排行榜** — 跨所有學界比賽的最佳時間排名，可按項目、性別、級別、賽季及組別篩選

### 改進（最新）
- **初賽及決賽時間** — 設有初賽及決賽的項目現在並排顯示兩個時間，晉級決賽的選手顯示兩輪成績，僅參加初賽的選手亦清楚標示
- **項目篩選** — 比賽成績現可按性別、級別、距離及泳式篩選，使用可點擊的標籤按鈕取代下拉式選單，篩選條件可自由組合

## 2026-03-21

### 改進（最新）
- **比賽頁面分頁功能** — 比賽列表現在每頁顯示20場，支援上一頁/下一頁瀏覽，不再一次載入全部
- **更快的比賽查詢** — 優化比賽列表查詢，頁面載入更快

### 新功能
- **我的隊伍** — 建立個人泳手關注名單，無需註冊帳號。從泳手個人頁面添加泳手，在隊伍頁面查看所有關注泳手的前5項個人最佳成績。可使用自訂代碼儲存和分享隊伍（例如 `/team?code=chan-family`）
- **排行榜：本地 vs 先進篩選** — 在排行榜上切換顯示全部、本地或先進賽事的成績
- **排行榜：性別、年齡組別及賽季篩選** — 按性別、年齡組別或賽季篩選排名，篩選條件可組合使用，切換項目時會自動保留
- **申訴功能** — 透過申訴頁面提交數據更正或報告缺失成績
- **比較功能** — 任意兩位泳手的個人最佳成績逐項對比
- **趨勢分析** — 進步最大的泳手及突破達標成績

### 改進
- **更快的頁面載入** — 所有頁面在載入數據時顯示骨架屏，泳會頁面載入速度大幅提升
- **年齡組別及賽季篩選**改為下拉式選單，介面更簡潔
- **泳手個人頁面** — 個人最佳成績表現在顯示比賽名次並連結至比賽頁面。互動式成績走勢圖展示所有比賽紀錄及最佳成績里程碑
- **泳會頁面** — 顯示完整泳會名稱（中英文），每季會員人數及獎牌數量圖表
- **比賽頁面** — 大型比賽支援按項目瀏覽
- **搜尋功能** — 顯示完整泳會名稱而非代碼

### 錯誤修復
- 修復泳會頁面出現重複泳手的問題
- 修復泳手個人頁面所有泳手顯示為「女子」的問題
- 修復比賽頁面在手機上的排版問題
- 修復部分游泳時間含有多餘字元的顯示問題

### 數據
- **402,563 項成績**，涵蓋 **16,701 位泳手**、**99 個泳會**及 **253 場比賽**（2014–2026）
- **65,843 項學界成績**，涵蓋 **13,275 位學生**及 **270 間學校**（2015–2025）
- 包括本地賽事、先進賽事、地區賽事（18區）、公開水域賽事及學界校際賽事
- 可搜尋游泳時間、分段成績、達標標準及名次
